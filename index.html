Data Quality Assessment

This analysis evaluates the data quality of an airport and heliport database for Argentina, essential for aviation safety, operational efficiency, and planning. The dataset includes fields like airport names, IATA/ICAO codes, geographic coordinates, elevation, and operational status. The assessment focuses on key dimensions:

  • Completeness: Reviewing missing values, particularly in elevation and GPS code fields.
  • Uniqueness: Detecting and addressing duplicate entries.
  • Accuracy: Cross-validating location data with external sources; some data points were initially mislocated over water bodies.
  • Consistency: Ensuring coherence across fields and identifying anomalies, particularly in elevation values.
  • Validity: Verifying that values fall within expected ranges for aviation data standards.

The report provides actionable recommendations for data cleaning and enhanced validation to support data quality improvements within the aviation sector.
